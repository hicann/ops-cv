## 功能说明

- 算子功能：对RGB图像的饱和度进行调整。

## 参数说明

| 参数名 | 输入/输出/属性 | 描述 | 数据类型 | 数据格式 |
|-----|----------|----------------------------------------------------------------------------------------------------------------------------------------------------|----------------|------|
| images   | 输入       | 输入Tensor，至少为三维，最后一维大小必须为3，按RGB三通道图像解释。 | FLOAT16、FLOAT | ND   |
| scale   | 输入       | 包含1个元素的一维Tensor，其唯一元素表示饱和度缩放因子。 | FLOAT          | ND   |
| y   | 输出       | 输出Tensor，shape和数据类型与images相同。 | FLOAT16、FLOAT | ND   |

## 约束说明

- images至少为三维，且最后一维必须为3。
- scale必须为仅包含1个元素的FLOAT Tensor。
- y的数据类型和数据大小必须与images一致。
