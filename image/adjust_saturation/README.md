## 功能说明

- 算子功能：对 RGB 图像的饱和度进行调整。

## 参数说明

| 参数名 | 输入/输出/属性 | 描述 | 数据类型 | 数据格式 |
|-----|----------|----------------------------------------------------------------------------------------------------------------------------------------------------|----------------|------|
| images   | 输入       | 输入 Tensor，至少为三维，最后一维大小必须为 3，按 RGB 三通道图像解释。 | FLOAT16、FLOAT | ND   |
| scale   | 输入       | 包含 1 个元素的一维 Tensor，其唯一元素表示饱和度缩放因子。 | FLOAT          | ND   |
| y   | 输出       | 输出 Tensor，shape 和数据类型与 images 相同。 | FLOAT16、FLOAT | ND   |

## 约束说明

- images 至少为三维，且最后一维必须为 3。
- scale 必须为仅包含 1 个元素的 FLOAT Tensor。
- y 的数据类型和数据大小必须与 images 一致。
