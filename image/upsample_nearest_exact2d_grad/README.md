## 功能说明

- 算子功能：[UpsampleNearest](../upsample_nearest/README.md)在exact_mode为true时的反向传播。
- 计算公式：
  
  $$
  gradInput(N, C, H, W) += gradOutput( N, C, ceil ( scales_h * H - 0.5 ),  ceil ( scales_w * W - 0.5 )) 
  $$
